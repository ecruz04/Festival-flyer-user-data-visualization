# Festival-flyer-user-data-visualization
My project is a Python-based app that allows Spotify users to create a personal festival flyer that is based on their listening history. This includes user's top artists and top songs. Using Flask and Spotipy API to create this project, the app transforms that data into a flyer. The flyer can be then downloaded to share it on social media platforms

How to use my program?
1. Run code on python (I used spyder)
2. Visit the link
3. Press login button and login using spotify
4. Flyer is displayed with multiple options. Choose between top songs or artists, download flyer to share, click on artist/song to be directed to the spotify page
5. Press log out
6. Done!
